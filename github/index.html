What is GitHub?

Posted by Jeongseop Yi (Patrick) on 2017-06-08

A collaborative development flatform for everyone


GitHub is a technological platform launched on April 10, 2008, by Tom Preston-Werner, Chris Wanstrath, and PJ Hyett. Their goal was to enable everyone to share their programming project and collaborate with others’ project as well. Its mission statement is “social coding”, and it provides various functions such as version control system, issue tracker, and feature request handler which help developers do collaborative programming. It now serves 22 million developers worldwide, 60 million repositories(projects) and 117 thousand businesses.
