McLuhan Theory: Hot and Cool Media

Posted by Jeongseop Yi (Patrick) on 2017-06-07

How GitHub is a “cool” media

  1. Software development was considered hot media because of the difficult skills it requires and closed development environment.
  2. Even though open source software development was emerging, there was no well-organized platform nor a tool that supported such a development environment.
  3. GitHub was created and started to support tools that can enable developers to create an open source project that can involve participation throughout the world.
  4. GitHub has served as a platform that users can participate freely to participate in various open source projects that they are interested in.
  5. GitHub is now a platform that represents open source development, boasting 22 million developers worldwide, 60 million repositories (projects) and 117 thousand businesses that use its platform due to the unique collaboration features that GitHub provides.
  6. GitHub has become a platform where developers all around the globe can contribute to a project and it has contributed to breaking down the initial barriers of programming: learning how to write code, review code, and participate.
