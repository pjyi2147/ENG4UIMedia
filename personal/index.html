Did GitHub changed the world?

Posted by Jeongseop Yi (Patrick) on 2017-06-08

Preface

In today’s modern society, technology has became an integral part of our lives. What powers those technologies used today is numerous softwares that are built by large number of developers made for general purposes. In those softwares, there are projects that are free to use, free to modify, and free to distribute; we call those projects open source projects. Open source projects existed even back in 1980s; however, these open source projects were scattered around the internet, and there was not a platform that everyone can freely create open source project in their interest, share their source and maintain the project by building communities of developers. What GitHub provides us is the platform that everyone can create open source project, share code, contribute to other projects, and even build a developer community that uses the project. Now, open source projects have become the backbone of today’s IT industry.

GitHub actually changed the world!

Github, as a platform, has provided us three benefits: reduction of time invested on developing, an open source community, and shrinkage of entry barriers of development.
