Did GitHub change the world?

Posted by Jeongseop Yi (Patrick) on 2017-06-08

Preface

In today’s modern society, technology has become an integral part of our lives. The powers that modern technologies have are numerous, and softwares are now being built for general purposes by a large number of developers. Among these softwares are projects that are free to use, free to modify, and free to distribute. We call those projects open source softwares. Open source projects existed even back in the 1980s; however, these open source projects were scattered around the internet, and there was not a platform that everyone could freely access to create open source projects based on their interest, share their sources or maintain the project by building communities of developers. What GitHub provides us is a platform wherein everyone can create open source projects, share code, contribute to other projects, and even build a developer community that uses the final project. Today, open source projects have become the backbone of the twenty-first century’s IT industry.

GitHub actually changed the world!

Github, as a platform, has provided us three benefits: reduction of time invested in developing, a collaboration of developers worldwide, and the diminishment of entry barriers of programming. GitHub has increased developers’ productivity by a large margin by hosting various open source projects and providing useful functions for programmers. For example, a static website generator Hexo was used to build this website, which is an open source project found on GitHub, which enabled me to build this website within 30 minutes.

<Hexo logo>

Furthermore, various IT companies use open source projects hosted on Github. To name a few, Twitter, Microsoft, and Facebook are all examples of such companies. By using pre-built open source projects, developers no longer need to develop from scratch, which they often had to do before GitHub. However, GitHub has provided developers an environment where they can reach their goals in more efficient ways, which is what enabled me to easily create a website using an open source project.


<React Native: cross-platform open source project by facebook used for instagram, facebook and various apps>

GitHub also changed how developers collaborate. Using functions that GitHub provides developers, such as function requests, error reports, and version control, it has become possible to build a single project that has more than a thousand collaborators, without having trouble managing the overall venture. For example, Facebook’s React Native has more than a thousand contributors around the globe; being one of the most popular application development frameworks. This made it possible to build projects that were previously considered impossible to develop since it required too much time and effort, and numerous applications we use today, including Facebook, Instagram, and etc, have been built using open source projects. Without GitHub, these open source projects would not exist, and it would have had a major negative impact on the digital life that we are enjoying now.


<React Native boasts over 1000 contributors, about 50k stars and 11.5k forks>

GitHub also lowers the barriers to development, which was previously considered to be too difficult to learn for those who were not computer science degree graduates. The platform has made it easy to participate in open source projects regardless of the ability to code. For example, there is always a need for translation. Bilingual enthusiasts can contribute to the project by providing translations. Beginners are included too, as the moderators often leave simpler issues for beginners to fix. This option for participation can arouse novice developers’ interest in open source development and provides chances for beginners through which they improve their development skills. The barriers to become a skillful programmer are lessening because of many online programming tutorials on the internet; however, those lessons end before the fledgling developers have been able to build real-world programming. The fact that GitHub helps these amateurs to participate in the actual program development helps them to improve their skills, and later this participation becomes an opportunity to create an open source program of their own, consequentially creating a positive loop for IT industries.



<It is not difficult to find translation/beginner tags in various repositories.>

GitHub changed the world serving as a platform for open source collaboration. It has drastically changed how developers build their applications, how they collaborate, and how they provide practical experience for new developers in the IT industry, which is the most important industry of the 21st century. How about joining GitHub today? You will learn more than you expected, and your participation will help to change the world.
