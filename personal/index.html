Did GitHub change the world?

Posted by Jeongseop Yi (Patrick) on 2017-06-08

Preface

In today’s modern society, technology has become an integral part in our lives. What powers our techonology today is numerous software that has been built by a large number of developers. In that software, there are projects that are free to use, free to modify, and free to distribute; we call them open source projects. The idea of open source projects existed even back in the 1980s; however, it was not popular among developers because of the scatterness of open source projects and the lack of platforms for those projects. What GitHub has provided is a platform where everyone can create open source project, share code, contribute to other projects, and even build a developer community. Now, open source projects have become the backbone of today’s IT industry through the support of GitHub.

GitHub actually changed the world!

Github, as a platform, has provided us three benefits: reduction of time invested in app development, collaboration of developers worldwide, and shrinkage of entry barriers of programming. GitHub has increased developers’ productivity by a large margin by hosting various open source projects and providing useful functions for programmers. For example, a static website generator Hexo was used to build this website, which is an open source project found on GitHub, which enabled me to build this website within 30 minutes.

<Hexo logo>

Furthermore, various IT companies uses open source projects hosted on Github. To name a few, Twitter, Microsoft, and Facebook uses pre-bulit open source projects to increase their productivity. By using pre-built open source projects, developers no longer need to develop from scratch, which they often had to do before GitHub. GitHub has provided developers an environment where they reach their goal in more efficient ways.


<React Native: cross-platform open source project by facebook used for instagram, facebook and various apps>

GitHub also changed how developers collaborate. Using functions provided, such as function requests, error reports, and version control, it became possible to build a project that more than a thousand developers around the globe can involve to a single project without having much trouble over the management of the project. For example, Facebook’s React Native have more than a thousand contributors around the globe; being one of the most popular app development frameworks. GitHub made it possible to build projects that were considered impossible to make through providing a system for collabortion for worldwide developers. Numerous application we use today, including Facebook, Instagram, and many other applications, are built using various open source project. Without GitHub, these open source projects would not exist, and it would have had a major negative impact on the digital life that we are enjoying now.


<React Native boasts over 1000 contributors, about 50k stars and 11.5k forks>

GitHub also lowered the barriers to development, which was considered difficult to learn for general public. The platform had made it easy to participate in open source project regardless of the ability to code. For example, there are always need for translation, or you can ask for a feature request. Also, for beginners, the moderators often leave easy issues to let the beginners fix them. This availability for the participation of projects can arouse the beginners to have some interest in open source development and provide chances for beginners through which they improve their development skills. The barriers to becoming a skillful programmer are is being diminished by many online programming lessons on the internet; however, those lessons often ends before the newbies are able to build real-world programming. The fact that GitHub helps novice developers to participate in the actual program development helps them to improve their skills. This participation later becomes an motivation to create an open source program on their own which becomes a positive loop for IT industries.



<It is not difficult to find translation/beginner tags in various repositories.>

GitHub changed the world serving as a platform and media of open source. It drastically changed how developers build their applications, how they collaborate, and how they train new developers into the IT industry. How about joining GitHub today? You will learn more than you expected, and the world will change with your participation.
